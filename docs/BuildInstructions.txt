Build BoxBoxF1Fantasy v2 from scratch

You are an expert Python data engineer and machine learning engineer.

You are tasked with building a production-quality Formula 1 fantasy prediction pipeline called BoxBoxF1Fantasy.

This system will:

Download Formula 1 telemetry and results data

Convert it into lap-level datasets

Engineer race performance features

Train predictive models

Generate predictions for upcoming races

Calculate expected fantasy points

Export predictions as JSON for a website

The system must be simple, deterministic, and robust.

Avoid overengineering.

No experimental spaghetti scripts.

Everything must follow the architecture defined below.

Core Design Principles

Follow these rules strictly.

1. Each script does one thing only

No script should exceed ~600 lines.

2. Scripts communicate via files only

Never rely on in-memory pipelines.

Every stage writes an output file used by the next stage.

3. Use Parquet for all datasets

Never use CSV unless unavoidable.

4. All configuration must live in a config folder

5. The pipeline must be reproducible

Same inputs → same outputs.

How It Works

📊

1. Data Collection

We collect free practice session data using FastF1 — lap times, sector times, tyre compounds, weather, and more.

⚙️

2. Feature Engineering

Raw lap data is processed into 27+ features per driver: pace metrics, consistency, tyre degradation, long-run performance, and more.

🤖

3. ML Predictions

XGBoost models trained on 2020-2026 historical data (2,600+ rows) combined with free practice telemetry — 136 features with walk-forward validation.

🏆

4. Fantasy Points

Predicted positions are converted into expected F1 Official Fantasy points, including overtakes, DNF risk, and fastest lap probability.

Model Details

  • Qualifying model: XGBoost (1200 trees, lr=0.025, depth=3) predicting from historical priors + FP telemetry
  • Race model: XGBoost (650 trees, lr=0.03, depth=5) with predicted quali as input
  • Two-layer features: Layer 1 = Jolpica historical priors (always available), Layer 2 = FP telemetry (when available)
  • Confidence: 0-100 based on data completeness and model agreement
  • Risk: Rolling 5-race DNF probability

Disclaimer

These predictions are for entertainment purposes only. F1 races are inherently unpredictable — weather, crashes, strategy calls, and reliability can dramatically alter outcomes. Use these predictions as one input for your fantasy decisions, not the only one.
